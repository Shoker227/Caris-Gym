# Caris-Gym (GoFit)

App de una sola página. **Todo el código (HTML, CSS y JS) está inline en `index.html`**:
no hay build, ni bundler, ni ficheros JS/CSS aparte. Backend: Supabase.
Se publica en GitHub Pages desde `main` (cada push es un deploy).

## Reglas al editar
- Modifica `index.html` en su sitio. NO lo dividas en varios ficheros ni añadas
  build tools, bundlers o frameworks salvo que te lo pida.
- No cambies la URL ni las claves de Supabase salvo petición expresa.
- No necesitas hacer commit ni push: se sube solo al terminar la respuesta.
